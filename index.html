MEAN Stack

MEAN is a open-source JavaScript software stack for building dynamic web apps. The MEAN stack comprised of MongoDB, Express.js, Angular and Node.js.

Above are four Links to help you get started and with bulding a MEAN stack web app. The first Link Full Stack MEAN tutorial shows you the basics on how to install all the neccessary parts for MEAN stack and how to create a web app from scratch. Hello-MEAN tutorial is a condensed tutorial to help you build a CRUD application using MEAN.The Hello-MEAN repo is the github repo that you can use to follow along, or you can clone the repo and run the project to use the web app.The staging area doc is a google doc that lists all the people who worked on this project and the process.There also is information on the MEAN stack.
